# NYCU Humanities Course Tool

陽明交通大學《人文科學中心》課程表下載工具。

提供 Streamlit 網頁介面，可自動：

- 抓取課程時間表資料
- 解析摘要分類
- 拆分 110 / 106 / 90 / 96 制
- 匯出 Excel

---

# 功能特色

- 自動抓取最新學年度課程
- 支援選擇學期
- 自動拆分摘要欄位
- Excel 美化格式
- 行政人員可直接使用

---

# 安裝方式

```bash
pip install -r requirements.txt
```

---

# 啟動方式

```bash
streamlit run app.py
```

---

# 欄位說明

| 欄位 | 說明 |
|---|---|
| 114學年後使用類別(110制) | 新制分類 |
| 109學年(含)以前適用(106制) | 舊制分類 |
| 舊制通識(90) | 舊版通識 |
| 舊制通識(96) | 舊版通識 |
| 摘要(原始) | 原始資料 |
